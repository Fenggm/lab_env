## VM Command Help
* Start the vm/image install
```
vagrant up
```
* Stop the vm
```
vagrant halt
```
* Update the vm
```
git pull && vagrant provision
```
* Suspend the vm
```
vagrant suspend
```
* Wake up the vm
```
vagrant resume
```
* Restart the vm
```
vagrant reload
```
